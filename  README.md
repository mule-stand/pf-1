# Проєкт ддя портфоліо

Це тестовий проєкт для портфоліо, з використанням таких технологій:
* HTML
* CSS
* JS
* TypeScript
* Docker
* Docker Compose
* Webpack
* React:
  * dsdddfd
  * sdfd
* Emotion JS
* Emotion JS

## Використання

Для запуску проекта з Docker та Docker Compose:
`make install`
`make up`
